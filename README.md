// SecureNote
A browser-based SPA for secure note taking

## What it does
-create, edit and save Notes
-Notes are stored locally in your browser using local storage
-No account or internet connection required

## How to use
Click New note to create a note
Click a note in the sidebar to edit

##tech stack
-html
-CSS
-JavaScript
